# Liri-Bot
A node command line

Type `node liri.js help` for a list of commands to use with this program

--this command `node liri.js spotify-this-song <your selected song>` will pull up info on your selected song

--this command `node liri.js movie-this <your selected movie>` will pull up info on your selected movie

--this command `node liri.js concert-this <your selected artist>` will pull up info on upcoming concerts for your selected artist

the <> symbols are not to be part of your input. 

new commands are appended onto the log.txt file
